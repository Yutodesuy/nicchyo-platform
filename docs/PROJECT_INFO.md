# プロジェクト情報
― 体制・歩み・貢献 ―

## プロジェクト概要

**nicchyo（ニッチョ）** は、高知・日曜市を舞台に、初めて訪れる来場者が日曜市特有の「迷い」を不安ではなく楽しさとして受け止め、安心して市を歩き始められる状態をつくることを目的とした、学生主体のデジタルプロジェクトです。

初来訪者は「何がどこにあるか」だけでなく、「何となく気になるが、うまく言葉にできない」といった曖昧な不安を抱えています。nicchyo は、**検索 → 相談（AI） → 会話（出店者）** という段階的な流れでこの不安を受け止め、体験へと橋渡しします。

## 体制・運営方法（部活動化）

nicchyo は単年度のプロジェクトではなく、日曜市という地域文化に継続的に関わるため、**部活動としての設立・運営**を前提としています。

### 部活動化の目的
1.  **継続的な知識・技術の継承**: 上級生から下級生へノウハウを引き継ぎ、活動を持続可能にする。
2.  **主体的・協働的な学びの場の創出**: 企画、開発、現地調査などを学生主体で行い、実社会と直結した学びを得る。
3.  **行政・地域との継続的な関係構築**: 高知市役所や出店者との信頼関係を長期的に築く。

### 運営体制
*   **主体**: 高知高専 nicchyo プロジェクト（re-KOSEN 採択事業 / 2025年度より部活動化予定）
*   **顧問**: 翁長 志保子 先生
*   **協力**: 高知市商業振興課街路市担当、日曜市出店者の皆様

## 地域・学校への貢献

nicchyo は、学生が主体的に地域文化に関わることで、多面的な価値を生み出しています。

### 地域社会への貢献
*   **文化の継承と発信**: 若い世代の視点で日曜市の魅力を再解釈し、デジタルを通じて可視化する。
*   **出店者支援**: 店舗情報発信の負担を軽減し、新規来場者との接点を創出する。
*   **課題解決**: 来場者や出店者の声を分析し、具体的な仕組みとして解決策を提案する。

### 学校への貢献
*   **実践的な学び**: 企画力、分析力、開発技術を実社会の中で身につける。
*   **地域連携**: 行政や地域と連携し、開かれた学校づくりに貢献する。

## 活動実績・これまでの歩み

nicchyo は、アイデア先行ではなく、現場との対話と実証を重ねて進めてきました。

*   **2025年7月**: 高知市との第1回協議（現状ヒアリング・構想説明）
*   **2025年8月**: re-KOSEN 採択・キックオフ合宿（プロジェクト正式始動）
*   **2025年9月**: 高知市との第2回協議（方針のすり合わせ）
*   **2025年10月**: 来場者・出店者アンケート実施（現場の声の収集）
*   **2025年11月**: 中間報告会（コンセプト・方向性の確認）
*   **2026年1月**: 最終報告会（予定・成果共有）

## 今後の展望・ロードマップ

nicchyo は完成形を固定せず、運用と改善を繰り返しながら育てていくサービスです。

### 1年目の目標：実運用と検証
*   デジタルマップの本格運用開始
*   「日曜市に合っているかどうか」の検証
*   フィードバックの収集と改善

### 2〜3年目の目標：定着と高度化
*   利用データに基づくサービスの質向上
*   日曜市を訪れる際の自然な選択肢としての定着
*   ネイティブアプリによる体験拡張の検討

### 将来的な広がり
*   得られた知見やモデルを、他のローカルマーケットや地域イベントへ応用する可能性も視野に入れています。ただし、拡大自体が目的ではなく、日曜市の魅力を次世代へ手渡すための土台づくりを最優先とします。
